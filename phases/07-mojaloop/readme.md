## Set DNS

No public domain name is used. So CoreDNS must be set resolve the local domain names of each sim-fsp

```json
    hosts {
      3.8.122.104 sim-sim-fsp201.local
      18.130.31.179 sim-sim-fsp202.local
      fallthrough
    }
```
